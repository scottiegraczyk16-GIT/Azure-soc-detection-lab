# Event Generator

A Flask application developed on an Azure Linux VM that generates simulated system events and appends them to a log file.

## Purpose

This application was built to simulate a simple event log source that can be used for monitoring and investigation exercises.

## Skills Demonstrated

- Python
- Flask
- Linux file handling
- Event log generation
- Web application deployment

## Log File

Messages are written to:

/home/azureuser/system_events.log
